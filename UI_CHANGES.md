## Key UI Changes

### 1. Form Changes
```
ADDED:
✅ Email field (required)
   - Type: email
   - Autocomplete: email
   - Validation: required

REMOVED:
❌ Shuffle button (not needed with paywall)

CHANGED:
🔄 "Get reading" → "Get reading summary"
```

### 2. Result Display
```
BEFORE:
- Full reading shown inline
- All rune details visible
- Copy/Print buttons

AFTER:
- Summary only (1-2 sentences)
- "Pay to unlock" button
- Hint about email delivery
```

### 3. New Pages
```
ADDED:
✅ /asteria/thanks
   - Post-payment confirmation
   - Instructions to check email

✅ /reading/{token}
   - Separate full reading page
   - Clean, printable layout
   - No navigation (focused on content)
```
